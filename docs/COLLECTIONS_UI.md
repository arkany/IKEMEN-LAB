## Design System

### Colors
- Background: zinc-950 (#09090b)
- Card Background: zinc-900/20 (transparent)
- Border: white/5 (subtle)
- Text Primary: white (#ffffff)
- Text Secondary: zinc-400
- Text Tertiary: zinc-500

### Typography
- Headers: Montserrat, 600 weight
- Body: System, 400 weight
- Captions: System, 400 weight, smaller size

### Icons
- Collections icon: folder.fill (SF Symbol)
- Size: 14pt for nav items
- Color: zinc-400 (default), white (hover/selected)

### Interactions
- Hover: Border changes from white/5 to white/10
- Click: Visual press feedback (scale 0.98)
- Cursor: Pointing hand on interactive elements
